Datenübertragung

Die ist üblicherweise die Hauptanwendung von Secure FTP. Es gibt drei Arten von Datenübertragungen: Herunterladen, Hinaufladen und Inhaltsangabe eines Verzeichnisses.

Inhaltsangabe eines Verzeichnisses

Die Inhaltsangabe eines Verzeichnisses geschieht üblicherweise automatisch. D. h. wenn Sie das Verzeichnis wechseln wird versucht, dessen Inhalt anzuzeigen. Sie können die Inhaltsangabe eines Verzeichnisses auch erzwingen, indem Sie den Aktualisieren-Knopf drücken, der sich in der Werkzeugleiste befindet.

Herunterladen

Herunterladen bedeutet, dass Sie eine Datei von einem anderen Rechner auf Ihren herunter übertragen. Dafür gibt es mehrere Möglichkeiten. Die einfachste ist, alle gewünschten Dateien auszuwählen und den Herunterladen-Knopf zu drücken (zu finden in der Werkzeugleiste). Damit übertragen Sie die ausgewählten Dateien in Ihr lokales Verzeichnis (welches auf der linken Seite des Fensters angezeigt wird).

Optional können Sie die ausgewählten Dateien vom Ziel-Verzeichnis (rechte Seite) in Ihr lokales Verzeichnis (linke Seite) ziehen. Wenn Sie diese auf ein Verzeichnis innerhalb Ihres lokalen Verzeichnisses ziehen wird dieses Unterverzeichnis ausgewählt was bedeutet, dass die heruntergeladenen Dateien dort gespeichert werden (anstatt im übergeordneten Verzeichnis).

Hinaufladen

Hinaufladen bedeutet, dass Sie eine Datei von Ihrem Rechner auf einen anderen hinauf übertragen. Dafür gibt es mehrere Möglichkeiten. Die einfachste ist, alle gewünschten Dateien auszuwählen und den Hinaufladen-Knopf zu drücken (zu finden in der Werkzeugleiste). Damit übertragen Sie die ausgewählten Dateien in das Ziel-Verzeichnis (welches auf der rechten Seite des Fensters angezeigt wird).

Optional können Sie die ausgewählten Dateien von Ihrem lokalen Verzeichnis (linke Seite) in das Ziel-Verzeichnis (rechte Seite) ziehen. Wenn Sie diese auf ein Verzeichnis innerhalb des Ziel-Verzeichnisses ziehen wird dieses Unterverzeichnis ausgewählt was bedeutet, dass die heruntergeladenen Dateien dort gespeichert werden (anstatt im Zielverzeichnis). Ebenso können Sie Dateien aus Ihrem Dateisystem (z. B. dem Schreibtisch) auf das Ziel-Verzeichnis ziehen.
