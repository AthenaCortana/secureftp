Transfert de données

Il s'agit assurément de la principale fonctionnalité de Secure FTP. Les types de transfert de données sont au nombre de trois: chargements, téléchargements et listages de répertoires.

Listage de répertoire

Les listages de répertoires sont habituellement effectués de manière implicite. En d'autres termes, quand vous changez de répertoire le logiciel tente de dresser la liste du contenu du dossier. Vous pouvez cependant forcer le listage d'un répertoire en pressant le bouton Actualiser, qui se trouve dans la barre d'outils.

Téléchargement

Effectuer un téléchargement signifie transférer un fichier depuis la machine de quelqu'un d'autre vers votre machine. Il y a plusieurs méthodes pour accomplir cette tâche. L'option la plus simple consiste à sélectionner tous les fichiers qui vous intéressent et à presser le bouton de téléchargement (qui se trouve dans la barre d'outils). Les fichiers sélectionnés seront alors téléchargés dans votre répertoire courant (que vous trouverez dans la partie gauche de la fenêtre).

Vous pouvez aussi faire glisser les fichiers sélectionnés depuis le côté correspondant à la machine distante (à droite) et les déposer du côté local (à gauche). Si vous faites glisser les fichiers vers un répertoire local, ce dernier sera mis en surbrillance, afin de signaler que les fichiers téléchargés seront enregistrés dans ce répertoire (et non dans le répertoire courant).

Chargement

Effectuer un chargement signifie transférer un fichier depuis votre machine vers la machine de quelqu'un d'autre. Il y a plusieurs méthodes pour accomplir cette tâche. L'option la plus simple consiste à sélectionner tous les fichiers que vous voulez transférer et à presser le bouton de chargement (qui se trouve dans la barre d'outils). Les fichiers sélectionnés seront alors chargés vers le répertoire courant distant (indiqué dans la partie droite de la fenêtre).

Vous pouvez également faire glisser les fichiers sélectionnés depuis le côté local (à gauche) et les déposer sur le côté distant (à droite). Si vous faites glisser les fichiers vers un répertoire distant, ce dernier sera mis en surbrillance, afin de signaler que les fichiers chargés seront enregistrés dans ce répertoire (et non dans le répertoire courant). Vous pouvez aussi accomplir cette tâche en faisant glisser des fichiers depuis votre système de fichiers (par exemple depuis votre bureau) vers le côté distant.
