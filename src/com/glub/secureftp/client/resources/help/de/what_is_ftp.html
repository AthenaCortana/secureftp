Was ist FTP?

FTP, oder das File Transfer Protocol (Dateiübertragungsverfahren), ist ein Verfahren, dass es Benutzern erlaubt, Dateien zwischen ihrem eigenen Rechner und jedem Rechner, den sie über ein Netzwerk erreichen können, auszutauschen.

Mit anderen Worten haben Sie so eine Möglichkeit, Dateien von Ihrem Rechner auf den eines Anderen oder von seinem auf Ihren Rechner zu übertragen.
