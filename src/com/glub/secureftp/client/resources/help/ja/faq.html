よくある質問

削除したファイルを復元できますか？

手短に言えば、できません。リモートサーバには、ごみ箱はありませんので、ファイルを削除すると、ただちに消去されます。ローカルでは、 ファイルの復元がサポートされているシステムも存在しますが、Secure FTP は、独立したプラットフォームですので、ファイルの復元機能は、ご用意できません。

NAT のサポート

Network Address Translation (NAT) は、頻繁に使用されるようになってきました。ファイヤーウォールや、たくさんの IP が使用できるデバイスを利用されている場合、NAT をご利用の方が多いでしょう。NAT と Secure FTP をご利用の方で、パッシブモードでの接続を使用されていない方は、注意が必要です。NAT を使用し、暗号化を有効にして接続している場合は、パッシブモードを使用して接続するか、アクティブ (PORT) データ転送と共に CCC (Clear Command Channel) を有効にしてください。有効にしない場合は、データ転送に失敗する恐れがあります。これは、暗号化を無効にして FTP に接続する際、NAT デバイスが FTP コマンドを実行し、アドレスルーティングを訂正しないからです。暗号化を有効にして接続している場合、NAT デバイスは正しい処理を行います。パッシブモードを使用して接続している場合、NAT は、この動作を確認する必要がなくなります。
