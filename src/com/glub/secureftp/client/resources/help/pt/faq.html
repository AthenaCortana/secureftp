Perguntas Frequentes

Posso recuperar ficheiros apagados?

Numa palavra, não. Quando se estão a eliminar ficheiros do servidor remoto, não há caixote de lixo ou caixa de reciclagem, assim os ficheiros são apagados imediatamente. Localmente, o seu sistema poderá suportar restauração de ficheiros, mas dado que o Secure FTP é independente da plataforma, não temos hipótese de fornecer essa funcionalidade.

Suporte Para NAT

NAT, ou "Network Address Translation" (que significa Tradução de Endereço de Rede), está a tornar-se mais frequentemente usada. Se estiver a utilizar uma barreira de protecção ("firewall") ou outro dispositivo que lhe dá vários IPs a partir de um, a hipótese é estar a usar NAT. A única coisa que tem de observar, quando estiver a usar NAT e o Secure FTP, é quando não estiver a usar ligação do tipo passivo. Se estiver por trás de NAT e está a efectuar uma ligação segura, terá de fazer uma de duas coisas: ligação passiva, ou, permitir a opção Canal de Comando Visível juntamente com uma ligação de dados (PORTA) activa, de outro modo as suas transferências de dados irão falhar. Isso ocorre porque durante uma ligação FTP não-segura o seu dispositivo NAT realmente "escuta" os comandos FTP e faz o endereçamento correcto. Mas, quando a ligação é segura, o dispositivo NAT não consegue penetrar e fazer o seu trabalho correctamente. Quando se fazem ligações passivas, o NAT não terá de se preocupar com este comportamento.
