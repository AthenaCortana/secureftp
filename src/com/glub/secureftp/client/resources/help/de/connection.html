Verbindungsaufbau über den Dialog Verbindung aufbauen


Sich mit einem Ziel-Rechner zu verbinden ist ein sehr einfacher Vorgang. Im einfachsten Falle benötigen Sie dafür nur grundlegendste Angaben wie Rechnername, Benutzername und Passwort.

Der Rechnername ist der Name des Rechner, mit dem Sie sich verbinden wollen. Der Benutzername ist der Name mit dem Sie sich auf diesem Rechner anmelden. Das Passwort ist der Schlüssel, der Ihren Benutzernamen bestätigt. In einigen Fällen können Sie sich anonym anmelden und haben dann weniger Rechte als ein ordentlicher Benutzer.

Standarmässig versucht Secure FTP eine Verbindung mit offener Sicherheit zu erstellen. Wenn diese Verbindung scheitert meldet dies Secure FTP und bietet eine ungesicherte Verbindung an. Achtung: Verwenden Sie diese Option mit Vorsicht. Wenn Sie dies zulassen sind Ihr Passwort und Ihre Daten öffentlich einsehbar.

Im Dialog Verbindung öffnen können Sie ein lokales und ein Ziel-Ausgangsverzeichnisse auswählen, in die Secure FTP nach erfolgter Anmeldung wechselt.

Ebenso können Sie den Verbindungstyp wählen. Vorgegeben ist eine passive Verbindung, was für die meisten Benutzer ausreicht. Das bedeutet, dass der Ziel-Rechner bei einer Datenübertragung entscheidet, wie Sie sich verbinden. Wenn Sie sich hinter einer NAT befinden werden Sie diese Option bevorzugen, oder eine offene SSL-Verbindund wird erstellt, bei der Sie die Option offener Befehlskanal wählen und die Option passive Verbindung abwählen, wenn dies der Ziel-Rechner unterst&uum;tzt.

Die Option fü den nicht-verschlüsselten Befehlskanal ist nur bei einer offenen SSL-Verbindung verfügbar. Nachdem sich der Benutzer verschlüsselt identifiziert hat wird der Befehlskanal (oder Kontrollkanal) wieder in eine unverschlüsselte Verbindung verwandelt. Diese Option gibt Benutzern hinter einer (NAT) Firewall die Möglichkeit, aktive (oder PORT-) Datenverbindungen zu verwenden.

Wenn Sie diese Optionen in Ihren Lesezeichen speichern können Sie sie über das Lesezeichen-Menü oder die Option Profile im Dialog Verbindung aufbauen erreichen. Weitere Informationen über Lesezeichen erhalten Sie hier.
