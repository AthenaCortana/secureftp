「ホストに接続」ダイアログボックスから接続する方法


サーバへ接続する方法は、とても簡単です。ホスト名、ユーザ名、パスワード等の基本的な情報だけ必要になります。

ホスト名は、接続先のコンピュータの名前です。 ユーザ名は、コンピュータへログインする際に使用する名前です。 パスワードは、ご使用のユーザ名でログインする為のキーです。 匿名のユーザ名、anonymousでログインすることもでき、これは、標準のユーザよりも権限が少なくなります。

初期設定では、SSL (Explicit) を使用して接続するように設定されています。接続が確立されなければ、セキュア接続を無効にして接続するかどうか通知されます。
注意： このオプションを使用するかどうかは、慎重に選択してください。セキュア接続を無効にすると、パスワードやデータが開放されることになります。

「ホストに接続」ダイアログボックスでは、接続が確立された際に表示する、リモートの開始フォルダと ローカルの開始フォルダが指定できます。

接続の種類の設定もできます。初期設定では、多くのユーザに使用されているパッシブモードで接続するように設定されています。これは、データを転送する際、サーバによって接続先が決定されます。ファイヤーウォール (NAT) を使用して接続する場合は、このオプションを選択してください。サーバでサポートされている必要がありますが、SSL (Explicit) を使用して接続する場合は、CCC (Clear Command Channel) を選択し、パッシブモードを使用して接続するオプションをオフにしてください。

CCC (Clear Command Channel) の使用は、SSL (Explicit) を使用して接続する場合のみご利用頂けます。コマンド（コントロール）チャンネルは、ユーザ認証後、暗号化を無効にした接続に復帰します。 このオプションは、ファイヤーウォール（NAT）を経由して、アクティブ（PORT）データ接続を可能にします。

これらのオプションは、ブックマークに保存すると、「ブックマーク」メニューや「ホストに接続」 ダイアログボックスにあるプロファイルオプションから利用できます。詳細は、ブックマークをご覧ください。
