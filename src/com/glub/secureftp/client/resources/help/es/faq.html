Preguntas habituales

¿Puedo recuperar los archivos eliminados?

En términos generales, la respuesta es no. Cuando se eliminan archivos de un servidor remoto, no hay ningún tipo de papelera de reciclaje, así que los archivos se borran inmediatamente. Localmente, su sistema puede soportar la recuperación de archivos, pero como Secure FTP es una plataforma independiente no hay modo de que nosotros podamos ofrecer esa funcionalidad.

Soporte para NAT

NAT, o Traducción de Dirección de Red, se está extendiendo rápidamente. Si usa un cortafuegos u otro dispositivo que le dé varias direcciones IP a partir de una, entonces usted está usando NAT. Lo único que usted necesita revisar cuando utiliza NAT y Secure FTP es cuando no está usando un tipo de conexión pasiva. Si estuviera utilizando NAT y haciendo una conexión segura, necesitará realizar una de las siguientes opciones: una conexión pasiva o habilitar la opción Canal de Comando Clear con una conexión de datos (PUERTO) activa, sino sus transferencias de datos fallarán. Esto se debe a que durante una conexión FTP no segura, su dispositivo NAT escucha comandos FTP y realiza el enrutamiento de direcciones correctas. Sin embargo, cuando la conexión es segura, el dispositivo NAT no puede realizar las comprobaciones y realizar la acción correcta. Al realizar conexiones pasivas, NAT no deberá preocuparse de este comportamiento.
