Navigation

Die Navigation in der Benutzerschnittstelle von Secure FTP soll einfach und Ihrem gewohnten Arbeitsfluss ähnlich sein.

Gebrauch der Benutzerschnittstelle


Verwendung von "Drag And Drop"


Um in der Ansicht der lokalen Verzeichnise zu navigieren gibt es eine schnellere Methode. Wenn Sie einen Ordner oder eine Datei in das linke Fenster von Secure FTP ziehen wechseln Sie damit in diesen Ordner bzw. in den Ordner, der diese Datei enthält. Eine schnelle Methode, sich auf Ihrer lokalen Festplatte zu bewegen.
