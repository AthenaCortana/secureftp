Navigation

La navigation dans l'interface Secure FTP a été pensée avec un but de simplicité et de similarité avec la philosophie du système que vous utilisez.

Utilisation de l'interface utilisateur


Utilisation du "Glisser - Déposer"


Il existe une méthode plus rapide pour changer de répertoire dans le panneau locale des répertoires. Si vous faites glisser un fichier ou un répertoire dans le panneau gauche (votre ordinateur), vous allez vous retrouver dans ce répertoire (ou le répertoire qui contient le fichier). Il s'agit seulement d'une méthode rapide pour naviguer dans votre disque local.
