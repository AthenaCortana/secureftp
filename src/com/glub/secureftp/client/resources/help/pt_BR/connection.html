Como conectar-se Via Diálogo de Abertura de Conexão


Conectar-se a um servidor é um processo bastante simples. No mínimo, precisará conhecer algumas informações básicas, incluindo o nome do servidor (sistema hospedeiro ou "host"), o nome de usuário e a senha.

O nome do servidor é o nome da máquina a que pretende conectar-se. O nome de usuário é o nome que vai utilizar para efetuar a entrada no sistema ("login") dessa máquina. A senha é a chave que permite que o seu nome de usuário entre no sistema. Nalguns casos poderá entrar no sistema anonimamente, caso em que poderá ter menos privilégios que um usuário normal.

Por defeito, nós tentaremos fazer uma conexão SSL explícita. Se esta conexão falhar, será notificado e ser-lhe-á oferecido efetuar uma conexão insegura. Nota: Deverá escolher esta opção com cuidado. Se permitir esta conexão, a sua senha e os seus dados ficam publicamente expostos.

No diálogo de Abertura de Conexão poderá também especificar uma pasta remota e uma pasta local de início, para as quais o programa irá, uma vez obtida entrada com sucesso no sistema remoto.

Poderá também definir o tipo de conexão. Por defeito é adotado o tipo de conexão passivo, que para a maioria dos usuários será suficiente. O que isto significa é que quando é feita uma transferência de dados, o servidor decidirá onde o usuário se irá conectar. Se estiver atrás de uma barreira de proteção ("firewall") NAT, vai querer ter esta opção selecionada ou, se o sistema suportar, será estabelecida uma conexão SSL explícita, selecione a opção Canal de Comando Visível e a opção passivo não selecionada.

A opção Canal de Comando Visível está disponível apenas quando se faz uma conexão SSL explícita. Depois de o usuário ser autenticado seguramente, o canal de comando (ou controle) é revertido para anular a conexão. Esta opção permite aos usuários atrás de uma barreira de proteção ("firewall") (NAT) conseguirem que conexões de dados ativas (Porta), funcionem.

Se salvar estas opções nos seus marcadores poderá aceder-lhes através do menu Marcadores ou da opção Perfil no diálogo de Abertura de Conexão. Poderá obter mais informações acerca de Marcadores aqui.
