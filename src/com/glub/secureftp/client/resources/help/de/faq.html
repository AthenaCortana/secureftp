Fragen & Antworten

Kann ich gelöschte Dateien wiederherstellen?

Kurz und bündig: nein. Auf dem Zielrechner Dateien gibt es keinen Papierkorb oder Mülleimer, so dass Dateien sofort gelöscht werden. Ihr Betriebssystem auf Ihrem Rechner mag das Wiederherstellen beherrschen, aber da Secure FTP plattform-unabhängig ist können wir diese Funktion nicht bieten.

Unterstützung für NAT

NAT (Network Address Translation), oder Netzwerk-Adress-Umsetzung, wird immer öfter eingesetzt. Wenn Sie eine Firewall oder eine andere Vorrichtung verwenden, die Ihnen statt einer IP deren viele zur Verfügung stellt, stehen die Chancen gut, dass Sie NAT einsetzen. Wenn Sie NAT und Secure FTP verwenden müssen Sie beachten, ob Sie eine passive Verbindungsart einsetzen. Wenn Sie jenseits einer NAT sind und eine gesicherte Verbindung aufbauen wollen müssen Sie entweder eine passive Verbindung wählen oder die Option für den Clear Command Channel zusammen mit einer aktiven (PORT) Verbindung einschalten. Sonst wird die Dateiübertragung scheitern. Dies geschieht, weil Ihre NAT-Einrichtung während einer unsicheren FTP-Verbindung auf FTP-Befehle achtet und Adressen korrekt weiterleitet. Wenn die Verbindung aber gesichert ist kann NAT nicht mitlauschen und korrekt arbeiten. Wenn die Verbindung passiv ist ist NAT nicht betroffen.
