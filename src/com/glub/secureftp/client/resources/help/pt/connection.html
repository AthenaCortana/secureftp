Como ligar-se Via Diálogo de Abertura de Ligação


Ligar-se a um servidor é um processo bastante simples. No mínimo, precisará conhecer algumas informações básicas, incluindo o nome do servidor (sistema hospedeiro ou "host"), o nome de utilizador e a palavra-passe.

O nome do servidor é o nome da máquina a que pretende ligar-se. O nome de utilizador é o nome que vai utilizar para efectuar a entrada no sistema ("login") dessa máquina. A palavra-passe é a chave que permite que o seu nome de utilizador entre no sistema. Nalguns casos poderá entrar no sistema anonimamente, caso em que poderá ter menos privilégios que um utilizador normal.

Por defeito, nós tentaremos fazer uma ligação SSL explícita. Se esta ligação falhar, será notificado e ser-lhe-á oferecido efectuar uma ligação insegura. Nota: Deverá escolher esta opção com cuidado. Se permitir esta ligação, a sua palavra-passe e os seus dados ficam publicamente expostos.

No diálogo de Abertura de Ligação poderá também especificar uma pasta remota e uma pasta local de início, para as quais o programa irá, uma vez obtida entrada com sucesso no sistema remoto.

Poderá também definir o tipo de ligação. Por defeito é adoptado o tipo de ligação passivo, que para a maioria dos utilizadores será suficiente. O que isto significa é que quando é feita uma transferência de dados, o servidor decidirá onde o utilizador se irá ligar. Se estiver atrás de uma barreira de protecção ("firewall") NAT, vai querer ter esta opção seleccionada ou, se o sistema suportar, será estabelecida uma ligação SSL explícita, seleccione a opção Canal de Comando Visível e a opção passivo não seleccionada.

A opção Canal de Comando Visível está disponível apenas quando se faz uma ligação SSL explícita. Depois de o utilizador ser autenticado seguramente, o canal de comando (ou controle) é revertido para anular a ligação. Esta opção permite aos utilizadores atrás de uma barreira de protecção ("firewall") (NAT) conseguirem que ligações de dados activas (Porta), funcionem.

Se gravar estas opções nos seus marcadores poderá aceder-lhes através do menu Marcadores ou da opção Perfil no diálogo de Abertura de Ligação. Poderá obter mais informações acerca de Marcadores aqui.
